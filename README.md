# RNA-Seq Differential Expression Analysis in R

This repository contains RNA-seq differential expression and functional enrichment analysis workflows performed in R using DESeq2 and Bioconductor packages.

## Included Analyses

- Differential gene expression analysis with DESeq2
- PCA and MA plot visualization
- Volcano plot generation
- Heatmap visualization
- Gene Ontology (GO) enrichment analysis
- KEGG pathway enrichment analysis
- Pathway visualization with pathview

## Tools and Packages

- R
- DESeq2
- clusterProfiler
- EnhancedVolcano
- pheatmap
- tidyverse
- pathview
- org.Hs.eg.db

## Dataset

- GSE165308 RNA-seq dataset
